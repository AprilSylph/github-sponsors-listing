# Contributing

- Install Deno in your CLI:
  https://docs.deno.com/runtime/getting_started/installation/
- Install Deno in your IDE:
  https://docs.deno.com/runtime/getting_started/setup_your_environment/

| Command                                 | Description                   |
| --------------------------------------- | ----------------------------- |
| **`deno install`**                      | Install project dependencies  |
| **`deno lint`**                         | Perform code linting          |
| **`deno fmt`**                          | Apply code formatting         |
| **`PERSONAL_ACCESS_TOKEN=… ./main.ts`** | Test the project locally      |
| **`deno outdated --update`**            | Update outdated dependencies  |
| **`deno outdated --update --latest`**   | Upgrade outdated dependencies |
